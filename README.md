# HEENɅ - Adaptive Intelligence System

**Master your habits. Build your character. 100% Privacy. Open Source.**

An intelligent, privacy-first habit tracker that combines character progression, adaptive learning, and gamification into one beautiful experience.

##  Key Features

- **Character Progression**: 100 levels with evolving phases and customizable identity paths
- **Adaptive Protocols**: Smart task generation based on your performance and mood
- **Dream Vault**: Journalistic prompts with 30-day reflection montages
- **Duo Mode**: Sync progress with a friend and unlock bonuses together
- **Premium Cosmetics**: Beautiful themes to personalize your experience
- **100% Privacy**: All data stays on your device. No cloud, no tracking, no servers.
- **Offline First**: Works perfectly without internet connection
- **Installable**: Works as a web app on desktop and mobile

##  Privacy

This is an open-source application with **zero external dependencies for data**:
- All data stored in browser's localStorage
- No Supabase or external servers
- No IP tracking or device fingerprinting
- No analytics or event tracking
- Complete source code transparency

##  Progressive Web App

HEENɅ is installable as a web app:
- **Desktop**: Install from Chrome/Firefox browser menu
- **Mobile**: Add to home screen (iOS/Android)
- Works offline with service worker caching
- Native app-like experience

##  Built With

- React + TypeScript
- Vite (lightning-fast build)
- Zustand (state management)
